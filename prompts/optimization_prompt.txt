-------------------------------------------------

2. SECTION-TO-ISSUE MAPPING

   - Issue mentions "personalization" → Refine existing references, do NOT add new ones
   - Issue mentions "portfolio" or "link" → Add or improve portfolio link placement with context

   - Issue mentions "structure" or "paragraphs" → Add proper paragraph breaks (\n\n) between sections

3. WHAT TO PRESERVE
   - If subject line has NO issue → Leave it UNCHANGED
   - If opening paragraph has NO issue → Leave it UNCHANGED
   - If CTA has NO issue → Leave it UNCHANGED

   - Portfolio links that are already present (unless issue says to improve them)
   - Overall email structure and flow

4. CONSTRAINTS
   - Do NOT introduce new claims or benefits not in the original
   - Target 150-200 words with clear paragraph structure
   - Do NOT change the tone drastically (respect persona context)
   - Do NOT add spam trigger words (FREE, URGENT, GUARANTEED, LIMITED TIME, CLICK HERE)
   - Keep B2B professional standards
   - This is service outreach, NOT a job application
   - Use paragraph breaks (\n\n) to separate: Hook, Value Prop, Proof, CTA
   - CRITICAL: Your optimization must score EQUAL OR HIGHER than original

5. PORTFOLIO LINK HANDLING
   - If issue mentions missing portfolio link → Add ONE relevant link naturally
   - If portfolio link exists but issue says it's not contextualized → Add brief context
   - Format: "We recently completed [relevant project]: [link]"
   - Do NOT force multiple links unless issue specifically requires it
