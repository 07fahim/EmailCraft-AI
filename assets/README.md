# Assets Folder

Add your images here:

## Required Images

### Banner

- `banner.png` - Main banner for README header (recommended: 1200x400)

### Video

- `video-thumbnail.png` - YouTube video thumbnail (recommended: 1280x720)

### Screenshots (in `screenshots/` folder)

- `landing-page.png` - Homepage/landing page
- `generate-email.png` - Email generation result
- `batch-emails.png` - Batch processing view
- `dashboard.png` - Analytics dashboard

## Recommended Sizes

| Image           | Size      | Format |
| --------------- | --------- | ------ |
| Banner          | 1200x400  | PNG    |
| Video Thumbnail | 1280x720  | PNG    |
| Screenshots     | 1920x1080 | PNG    |

## Tips

1. Use high-quality screenshots
2. Crop to remove unnecessary UI
3. Compress images for faster loading
4. Consider using dark/light mode screenshots
